A Game of Chess
Created a chess game in Python using Pygame to display the board and move pieces, and created a basic bot to play against.
Project Details / Background
I created a chess game from scratch. My main motivation for this game is a facination with chess logic, so I added an AI bot to play against. It uses a minimax approach to move searches, and does alpha-beta pruning to optimize searches. Only searches a depth of 2 by default, but can be changed to any reasonable depth given enough time.
I am working on creating a move-ordering optimization to the bot, which should allow the alpha-beta pruning to be more effective. It will try and check moves that have a high likelyhood of being good moves first, so that it can prune more branches faster. This should have a substantial impact on speed once completed, and may allow for more depth in the minimax search.
